# Code of Conduct

This project expects respectful, professional collaboration.

Participants should:

- Use welcoming and inclusive language.
- Respect differing viewpoints and experiences.
- Accept constructive feedback gracefully.
- Focus on what is best for users and the project.

Unacceptable behavior includes harassment, discrimination, personal attacks, and
publishing private information without permission.

Report concerns privately to the maintainers listed in `OWNERS`.
